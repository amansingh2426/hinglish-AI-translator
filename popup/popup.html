📘 How to Use

  • Select any text on a webpage.
  • Right-click and choose:
    • “Translate to Hinglish” – for direct translation
    • “Explain in Hinglish” – for explanation using AI
  • The translated or explained text will show in a popup.
  • Your API key will be stored locally and securely.
